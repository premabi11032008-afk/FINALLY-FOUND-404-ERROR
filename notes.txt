# EcoMine AI - Features added

## Core Setup
- Initialized React + TypeScript app with Vite
- Added Tailwind CSS v4 for styling
- Configured dependencies: recharts, react-leaflet, lucide-react, react-router-dom, framer-motion

## Pages & Layout
- Layout components: Sidebar (NavLinks), TopBar (search + user info)
- Routing via BrowserRouter: / (StoryPage), /auth, /dashboard (index + /resale + /scrap)
- StoryPage — full-scroll storytelling landing page with interactive device evaluator
- AuthPage — login/signup form (navigates to /dashboard on submit)
- Dashboard — Analytics with Recharts LineChart + BarChart + FuturisticPredictionMap (Leaflet)
- ResaleExchange — AI decision form (condition slider → recommend Resell / Refurbish / Scrap)
- ScrapRecovery — bulk upload UI + live metal pricing table

## ML Backend (FastAPI)
- backend/main.py — FastAPI server, loads xgboost_model.pkl + realistic_dataset.csv on startup
- GET /api/predict?month=M&year=Y — returns predictions for 5 Indian regions + top_regions[]
- Regions: North, South, East, West, Central (India)
- Served with uvicorn on http://127.0.0.1:8000, CORS open

## ML Model Files
- xgboost_model.pkl — trained XGBoost regressor
- realistic_dataset.csv — historical monthly e-waste totals per region (2024-01 to 2025-12)
- get_prediction_from_model.py — prediction function (uses lag_1/2/3 + rolling_mean_3 features)
- ml_model_training.py — training script

## Frontend: Futuristic World Map (StoryPage Hero)
- WorldPredictionBackground.tsx — thin wrapper (pointer-events-none) used in StoryPage hero
- FuturisticWorldMap.tsx — main SVG map component:
    * 5 organic blob regions (West/North/Central/East/South as abstract shapes)
    * Blobs gently overlap using mix-blend-mode: screen for glowing merge effect
    * Fetches /api/predict dynamically — hotspots controlled by ML model output
    * Framer Motion: ping rings + pulsing core on active regions, smooth opacity transitions
    * Floating particles (32), SVG scan-line sweep, corner HUD brackets
    * Month/Year selector pill (‹ Jan 2026 ›) for testing — triggers live re-fetch on change
    * Loading indicator (···) while fetching

## Dashboard Map (separate)
- FuturisticPredictionMap.tsx — Leaflet map with CircleMarkers on India lat/lng coords
- Mounted in Dashboard.tsx under "Global Predictive E-Waste Assessment" section
