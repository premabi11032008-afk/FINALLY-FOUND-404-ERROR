# Re-Cover AI 2.0 — PROJECT NOTES
## Last updated: 2026-04-11 (Morning Session)

## 🏗️ Architecture Upgrade
- **Route Decoupling**: The "Resale & Exchange" (AI Assessment Matrix) has been decoupled from the Partner Dashboard.
  - `/` -> Mission-driven StoryPage.
  - `/resale` -> Standalone AI Assessment Matrix for consumer liquidity.
  - `/dashboard` -> Secure Partner Analytics & Scrap Recovery Hub.
- **Unified Auth**: All "Become a Partner" and "Bulk Liquidation" entry points now funnel through the central `/auth` portal.

## 🧠 Hybrid AI Evaluation Engine
- **Dedicated Presence**: Moved the evaluator to a distraction-free top-level page (`/resale`).
- **Conservative Pricing**: Multi-angle analysis identifies the most damaged view to ensure risk-free platform assessments.
- **Verification Stack**: YOLOv8 device detection + Gemini 1.5 Vision fallback + Human-in-the-loop confidence slider.

## 📊 Visual Intelligence (New)
- **Real-time Synchronization**: Frontend now fetches dynamic metrics from the backend `/api/stats` endpoint.
- **Impact Tracking**:
  - **Extraction Velocity**: Monthly area charts tracking e-waste processing in Tons.
  - **Recovery Mix**: Donut chart visualizing composition metrics (Gold, Copper, Plastic, Rare Earths).
  - **Path Distribution**: Comparative bar charts for Resale vs Recycling volumes.
- **Predictive Mapping**: Interactive Hotspot map showing regional e-waste generation projections.

## 💬 Intelligent Assistant Integration
- **Context-Aware Navigation**: Chatbot can now trigger automatic redirects to the standalone `/resale` page when users inquire about device value.
- **Backend Security**: Gemini API transactions are processed server-side to prevent client-side key exposure.

## 🛠️ Tech Stack Refinement
- **Frontend**: React 19, Vite, Tailwind CSS v4, Lucide Icons, Framer Motion, Recharts.
- **Backend**: FastAPI, Ultralytics (YOLOv8), Gemini 1.5 SDK, Pandas (Dataset Processing).

## 🚀 To Start the App
1. **Frontend**: `cd frontend && npm run dev`
2. **Backend**: `cd backend && python -m uvicorn main:app --host 127.0.0.1 --port 8000`
