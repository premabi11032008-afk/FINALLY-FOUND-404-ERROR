## ML Backend (FastAPI)
- backend/main.py — FastAPI server, loads xgboost_model.pkl + realistic_dataset.csv on startup
- GET /api/predict?month=M&year=Y — returns predictions for 5 Indian regions + top_regions[]
- Regions: North, South, East, West, Central (India)
- Served with uvicorn on http://127.0.0.1:8000, CORS open

## ML Model Files
- xgboost_model.pkl — trained XGBoost regressor
- realistic_dataset.csv — historical monthly e-waste totals per region (2024-01 to 2025-12)
- get_prediction_from_model.py — prediction function (uses lag_1/2/3 + rolling_mean_3 features)
- ml_model_training.py — training script

## Frontend: Futuristic World Map (StoryPage Hero)
- WorldPredictionBackground.tsx — thin wrapper (pointer-events-none) used in StoryPage hero
- FuturisticWorldMap.tsx — main SVG map component:
